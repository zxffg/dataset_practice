### bank_deposit
Проект: 
Анализ клиентов банка

Цель: 
Проверить факторы, влияющие на открытие депозита.

Датасет: 
[Bank Marketing Dataset](https://www.kaggle.com/datasets/prakharrathi25/banking-dataset-marketing-targets)

Основные выводы:
1. Клиенты с высшим образованием имеют более высокий медианный баланс по сравнению с другими группами.
2. Наличие кредитных обязательств может влиять на склонность клиента к открытию вклада.

Используемые технологии:
Python 3.14, Pandas 3.0.3, Matplotlib XXX, Jupyter (VS Code), uv 0.1.0

Запуск:
1. Клонирование репозитория
2. Установка зависимостей и библиотек. Используйте команду `uv sync`
